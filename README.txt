3D Game Hub - Upgraded (Style 2: Neon Cyber, Low-Poly Realistic)
--------------------------------------------------------------
Place real GLTF/GLB models into assets/models/ (runner.glb, player.glb, car.glb, enemy.glb, tree.glb)
Open index.html via a static server (python -m http.server) or upload to GitHub Pages.

To run locally (recommended):
1. Open PowerShell (or Terminal)
2. cd to this folder
3. python -m http.server 8000
4. Open http://localhost:8000 in your browser
